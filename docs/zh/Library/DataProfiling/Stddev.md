# Stddev

## 函数简介

本函数用于计算单列数值型数据的总体标准差。

**函数名：** STDDEV

**输入序列：** 仅支持单个输入序列，类型为 INT32 / INT64 / FLOAT / DOUBLE。

**输出序列：** 输出单个序列，类型为DOUBLE。序列仅包含一个时间戳为0、值为总体标准差的数据点。

**提示：** 数据中的空值、缺失值和`NaN`将会被忽略。
