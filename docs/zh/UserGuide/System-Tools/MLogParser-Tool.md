## Mlog解析工具

0.12.x版本之后，IoTDB将元数据文件编码成二进制。

如果想要解析元数据成人可读的方式，可以使用本工具来解析指定元数据文件。

该工具可以同时解析snapshot文件和mlog文件。

### 使用方式

Linux/MacOS
> ./mLogParser.sh -f /your path/mlog.bin -o /your path/mlog.txt

Windows

> .\mLogParser.bat -f \your path\mlog.bin -o \your path\mlog.txt
